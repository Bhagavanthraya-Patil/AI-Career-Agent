# templates/resumes

Base resume templates in multiple formats.

## Formats

- `standard.md` — Markdown template for LLM processing
- `ats_friendly.docx` — DOCX template optimized for ATS parsing
- `modern.tex` — LaTeX template for professional typesetting

## Variables

Templates use placeholder variables (e.g., `{{name}}`, `{{skills}}`, `{{experience}}`) that are replaced by the Resume Tailor Agent during generation.
