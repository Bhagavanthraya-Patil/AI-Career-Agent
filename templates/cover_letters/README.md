# templates/cover_letters

Base cover letter templates with customizable sections.

## Contents

- `standard.md` — General-purpose cover letter template
- `technical.md` — Template optimized for engineering/developer roles
- `executive.md` — Template for senior/leadership positions

## Variables

Templates use placeholders (e.g., `{{company}}`, `{{role}}`, `{{skills_match}}`) that are populated by the Cover Letter Agent.
