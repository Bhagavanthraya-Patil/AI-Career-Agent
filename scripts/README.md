# scripts

Utility scripts for development, deployment, and maintenance tasks.

## Planned Scripts

- `setup.sh` — Initial project setup (install dependencies, init database)
- `seed_data.py` — Populate database with sample data for development
- `run_agent.py` — CLI entry point for running individual agents
- `export_logs.py` — Export and archive agent activity logs
- `docker-compose.yml` — Container orchestration for local development
