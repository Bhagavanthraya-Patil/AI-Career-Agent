# backend/app

Core application package containing all backend logic.

## Submodules

- `agents/` — AI agent definitions, workflows, and orchestration
- `api/` — REST API routers, controllers, and middleware
- `core/` — Application configuration, security, logging, and utilities
- `db/` — Database models, migrations, and repositories
- `schemas/` — Pydantic request/response schemas
- `services/` — Business logic, LLM gateway, and external integrations
