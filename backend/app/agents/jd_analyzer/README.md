# backend/app/agents/jd_analyzer

JD Analyzer Agent — parses raw job descriptions and extracts structured metadata.

## Responsibilities

- Extract technical requirements, soft skills, responsibilities, and qualifications
- Identify required years of experience, core tools, and keyword tags
- Produce structured JSON output for downstream agents

## Input

Raw job description text

## Output

Structured JSON containing skills, experience requirements, tools, and keyword tags
