# backend/app/agents/cover_letter

Cover Letter Agent — generates tailored, professional cover letters aligned with the target role.

## Responsibilities

- Compose personalized cover letters referencing specific JD requirements
- Adapt tone and format based on company culture and role seniority
- Output in ATS-friendly plain text and formatted DOCX

## Input

User profile, target job details, tailored resume context

## Output

Tailored cover letter in plain text and DOCX formats
