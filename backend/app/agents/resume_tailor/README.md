# backend/app/agents/resume_tailor

Resume Tailor Agent — customizes the user's work history to align with target job descriptions.

## Responsibilities

- Rewrite bullet points to mirror JD terminology
- Re-rank and emphasize relevant skills
- Generate ATS-friendly markdown output for PDF/DOCX generation

## Input

Base user profile + target job skills/keywords from JD Analyzer

## Output

Customized markdown text representing tailored work history
