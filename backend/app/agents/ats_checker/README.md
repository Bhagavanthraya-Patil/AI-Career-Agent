# backend/app/agents/ats_checker

ATS Analyzer Agent — scores resume-JD compatibility and provides actionable improvement suggestions.

## Responsibilities

- Compute keyword density and semantic similarity between resume and JD
- Evaluate structural readability and ATS parsing friendliness
- Generate match score (0-100) with improvement recommendations

## Input

Tailored resume text + target job description

## Output

Match score (0-100) and a list of actionable improvement suggestions
