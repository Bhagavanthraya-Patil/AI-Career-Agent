# config

Configuration files and environment templates for the AI Career Agent.

## Contents

- `.env.example` — Template for required environment variables (API keys, database URL, etc.)
- `settings.yaml` — Application-level configuration defaults
- `logging.yaml` — Logging level and format configuration
- `agents.yaml` — Agent-specific parameters (temperature, max tokens, model selection)
- `scraping.yaml` — Scraping configuration (delays, retries, user agents)
