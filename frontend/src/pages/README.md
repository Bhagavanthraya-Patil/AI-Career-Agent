# frontend/src/pages

Page-level view components, each corresponding to a route.

## Planned Pages

- `Dashboard/` — Overview with stats, recent activity, quick actions
- `Profile/` — User profile management and skill editing
- `Jobs/` — Discovered jobs list with filtering and search
- `JobDetail/` — Single job view with analysis and actions
- `Resumes/` — Resume tailoring and preview
- `Applications/` — Application lifecycle tracker with status pipeline
- `Settings/` — System configuration and API key management
