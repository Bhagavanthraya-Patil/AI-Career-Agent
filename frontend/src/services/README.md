# frontend/src/services

API integration layer — typed clients for communicating with the backend.

## Planned Services

- `apiClient.ts` — Axios/fetch wrapper with base URL, auth, error handling
- `jobsService.ts` — Job CRUD endpoints
- `resumesService.ts` — Resume generation and management endpoints
- `applicationsService.ts` — Application tracking endpoints
- `agentsService.ts` — Agent trigger and status endpoints
